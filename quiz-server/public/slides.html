AI时代稀缺的六大能力

人才培养目标重塑

5 / 28

能力一：精准提问能力（Prompt思维）

💡 核心

AI只能回答问题，无法自主定义有价值的问题。好问题远比重构答案更值钱。

🛠️ 课堂实操

每节课留5分钟"好问题时间"；教学生用"身份+任务+要求+格式"模板向AI提问；评价不仅看"答得对不对"，更看"问得好不好"。

🤔 反思：我们平时是不是太喜欢"正确答案"了？有没有给过学生提问的空间？

6 / 28

能力二：原创创造能力

💡 核心

AI依靠过往数据做概率生成，只能复刻已有内容。人类独有的灵感、从零到一的突破性创新，机器无法复刻。

🛠️ 课堂实操

减少"标准答案"训练，增加"开放题"比重；鼓励学生"异想天开"；跨学科融合教学。

AI会拼接存量，人类要创造增量

7 / 28

能力三：审美与品位判断力

💡 核心

AI可以批量产出内容，但没有真实主观感受，分不清"高级"与"有温度"。审美是取舍、调性、人文质感的综合判断。

🛠️ 课堂实操

任何学科都有审美：数学的简洁之美、物理的对称之美、文学的意境之美；带学生赏析"好作品"的差异；鼓励创造"有美感"的作业。

8 / 28

能力四：价值决策能力

能力五：共情协作能力

能力六：长期愿景与韧性

价值决策：AI只有数据，无法判断伦理与长期利弊。课堂中引入"两难情境"讨论。

共情协作：AI无真实情感。增加小组合作，教学生"倾听"而非只是"表达"。

长期韧性：AI只处理当下任务。帮助学生建立成长型思维，正确看待失败。

🏔️ 偏远地区学生的"两手准备"

高考，我们要认真对待

对于偏远地区学生，高考依然是改变命运最重要的通道。我们不否定高考，而是要更好地应对高考。

但只有高考分数，是不够的

进了大学后，偏远地区学生往往"后劲不足"——只会刷题，不会自主学习。AI时代，这个问题会更突出。

既不放弃高考，又让学生"既考得好，也走得更远"

13 / 28

为什么"教师主导+学生主体"不能变？

🎯 教师主导不能变

价值观需要引导（不是放任自流）；课堂需要设计（不是随意发散）；学困生需要陪伴（AI没有温度）。

🎯 学生主体不能变

学习是学生自己的事（别人替代不了）；内驱力来自自主（不来自灌输）；AI是工具（用工具的人是学生）。

AI时代教师的四种身份

1️⃣ 学习设计师

设计学习路径、选择学习资源、搭建学习场景（首要身份）

2️⃣ 学情诊断师

用数据了解每个学生的真实状态，精准干预

3️⃣ 内驱力激发师

点燃学生的好奇心，让学生愿意学、喜欢学

4️⃣ 价值观引导师

AI不讲价值观，这件事只能老师来做

"三段四化"智慧课堂教学模式

📋 课前（前置学习）

教师设计前置学习单推送给学生；学生在平板完成自学质疑；AI收集学情数据。

📋 课中（精准教学）

不再"从头讲到尾"；重点讲学生不懂的、容易错的；分层教学。

📋 课后（个性化巩固）

AI根据每个学生情况推送不同练习题；错题自动整理；教师个别辅导。

CHAPTER FOUR

课堂教学怎么改？

具体可操作的方法

21 / 28

方法一：培养学生精准提问能力

🛠️ 每节课加入"好问题时间"（最后5分钟）

规则：不接受"是不是""对不对"类问题；鼓励学生提出：为什么？如果不这样会怎样？

🛠️ 用AI训练提问能力

让学生用AI回答问题，然后评价：AI的回答好不好？如果不好，是哪个问题没问清楚？

22 / 28

方法二：麻省理工学霸AI学习法

🧠 核心：三问法（先搭框架，再填细节）

第一问：这个领域专家共有的5个核心思维模型是什么？

第二问：专家们存在根本性分歧的3个地方是什么？

第三问：能区分"深度理解者"和"单纯记忆者"的10个问题是什么？

💡 课堂实操：不再"先讲定义再讲例题"，而是"先让学生用AI搭建知识框架，再来精细化理解"。

23 / 28

方法三：给孩子选择权，激发内驱力

🗝️ 核心：选择权 = 内驱力的源头

学习节奏让学生选；作业形式可以选；犯错的空间要留。

⚠️ 关键：选择权不等于放任自流

是在教师设计的框架内，给学生自主的空间。
