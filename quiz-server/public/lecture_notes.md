## 第二部分：人才培养目标重塑——AI时代稀缺的是什么

### 【核心内容】AI时代稀缺的六大能力（20分钟）

> **配套视频**：播放"AI时代稀缺的六大能力（黄仁勋体系）"视频片段
> **B站链接**：https://www.bilibili.com/video/BV1Q1DXBFEC4/
> **文章来源**：Todo工作室《AI时代稀缺的六大能力》

---

#### 能力一：精准提问能力（Prompt思维）

**核心讲解：**

AI只能回答问题，无法自主定义真正有价值的问题。未来，能提出好问题的人，远比能回答标准答案的人值钱。

**课堂实操建议：**
- 每节课留5分钟"好问题时间"：鼓励学生提出课本之外的问题
- 教学生用"身份+任务+要求+格式"的模板向AI提问
- 评价学生时，不仅看"答得对不对"，更看"问得好不好"

**教师反思：**
> 我们平时是不是太喜欢"正确答案"了？有没有给过学生提问的空间？

---

#### 能力二：原创创造能力

**核心讲解：**

AI依靠过往数据做概率生成，只能复刻、拼接已有内容。人类独有的灵感、从零到一的突破性创新，机器无法复刻。

**课堂实操建议：**
- 减少"标准答案"训练，增加"开放题"比重
- 鼓励学生"异想天开"，不直接否定"错误答案"
- 跨学科融合：语文老师可以聊科技，物理老师可以聊美学

**一句话总结：**
> AI会拼接存量，人类要创造增量。

---

#### 能力三：审美与品位判断力

**核心讲解：**

AI可以批量产出内容，但没有真实主观感受，分不清什么是"高级"、什么是"有温度"。审美是一种综合判断能力，是AI的盲区。

**课堂实操建议：**
- 任何学科都有审美：数学的简洁之美、物理的对称之美、文学的意境之美
- 带学生赏析"好作品"和"一般作品"的差异
- 鼓励学生创造"有美感"的作业，而不只是"正确"的作业

---

#### 能力四：价值决策能力

**核心讲解：**

AI只能输出客观数据，无法权衡伦理、长期利弊、人性取舍。在复杂模糊场景下做判断，是人类独有的能力。

**课堂实操建议：**
- 在课堂中引入"两难情境"讨论
- 引导学生思考：这件事"能做"不等于"应该做"
- 结合学科内容讨论科技伦理（如AI换脸、深度伪造）

---

#### 能力五：共情协作能力

**核心讲解：**

深度理解他人情绪、凝聚团队、化解矛盾——AI没有真实情感，这些都无法替代。

**课堂实操建议：**
- 增加小组合作学习，但要有明确的协作规则
- 教学生"倾听"而不只是"表达"
- 课堂中营造心理安全氛围：学生不怕犯错、敢于表达

---

#### 能力六：长期愿景与韧性

**核心讲解：**

AI只处理当下任务，不具备长远格局与抗压韧性。而人的成长，恰恰是一个长期的过程。

**课堂实操建议：**
- 帮助学生建立"成长型思维"：能力是可以提升的，不是固定的
- 教学生正确看待失败：错题是宝藏，不是耻辱
- 引导学生设立长期目标，而不只是"下次考试"

---

### 小结：从"知识容器"到"能力提升"（5分钟）

**PPT展示对比图：**

```
传统培养目标          →          AI时代培养目标
─────────────────────────────────────────────
知识记忆能力强      →     精准提问能力强
刷题速度快          →     原创创造能力强
标准答案命中率高    →     审美判断能力强
听话、守规矩        →     价值决策能力强
单打独斗            →     共情协作能力强
短期冲刺能力        →     长期韧性能力强
```

> **核心观点**：我们不是要放弃知识教学，而是在知识教学的基础上，更加注重能力的培养。知识是载体，能力是目标。

### 特别篇：偏远地区学生的"两手准备"（10分钟）

**讲师话术：**

> 说到这里，可能有的老师会想："这些道理我都懂，但我们的情况是——学生在偏远地区，资源比不上大城市，高考是当前最现实的出路，我哪有时间培养这些能力？"
>
> 这个问题，我问过自己很多次。今天，我想和大家分享我的答案。

**核心观点：高考 + 学习能力，两手都要硬**

1. **高考，我们要认真对待**
   - 对于偏远地区的学生，高考依然是改变命运最重要的通道
   - 我们不否定高考，而是要更好地应对高考

2. **但只有高考分数，是不够的**
   - 进了大学之后，来自偏远地区的学生往往"后劲不足"
   - 根本原因：只会刷题，不会自主学习
   - AI时代，这个问题会更加突出

3. **两手准备的具体路径**
   - 用AI赋能学习，让刷题更高效（麻省理工学霸AI学习法）
   - 在学习知识的同时，刻意培养六大能力
   - 哪怕每周只有一节课，专门做"能力训练"

> **一句话**：我们不放弃高考，但我们要让学生"既考得好，也走得更远"。

## 第三部分：教师角色再定位——主导与主体的辩证统一

### 澄清误区："教书给AI，教师只育人"对吗？（10分钟）

**讲师话术：**

> 现在有一种说法很流行："AI时代，教书交给AI，教师做好育人工作就行了。"
>
> 这句话，对了一半，也错了一半。
>
> 对的一半是：教师确实应该更加关注"育人"。
> 错的一半是：教书和育人，从来就不能分开。

**核心立场（板书强调）：**

```
教师主导  +  学生主体  =  好课堂

这两个，都不能变！
```

**为什么不能变？**

1. **教师主导不能变**
   - 价值观需要引导，不是放任自流
   - 课堂需要设计，不是随意发散
   - 学困生需要陪伴，AI没有温度

2. **学生主体不能变**
   - 学习是学生自己的事，别人替代不了
   - 内驱力来自自主，不来自灌输
   - AI是工具，用工具的人是学生

> **结论**：不是"教书给AI"，而是"用AI来教书，教师专注于育人 plus"——plus是更高层次的教书。

## 第四部分：AI赋能教学——偏远地区的弯道超车机会

### 现实困境：我们没有大城市的资源（5分钟）

**坦诚面对现实：**

- 偏远地区学校 vs 大城市名校，差距客观存在
- 优质师资、丰富课程、先进设备……我们都有差距
- 但AI时代，这一切正在发生改变

> **核心观点**：AI不是让强者更强，而是让弱者有机会追上。——因为AI把优质教育资源，以极低的成本，送到了每一个有网络的地方。

---

### "三师共育"模式：AI老师 + 同伴 + 真人教师（15分钟）

> **案例来源**：Todo工作室《安徽阜南亲情高级中学教学模式》《"小鹿爱学"AI平板自主学习+三师共育落地校》

**什么是"三师共育"？**

```
传统课堂：1个真人老师 → 面对50个学生
三师共育：
  - AI老师（24小时在线，个性化辅导）
  - 同伴互助（学生教学生，效果更好）
  - 真人教师（设计、引导、陪伴）
```

**"三师共育"在偏远地区的特殊价值：**

| 传统困境 | AI如何解决 |
|---------|-----------|
| 没有优质师资 | AI提供顶级教学资源 |
| 学生水平差异大，老师顾不过来 | AI个性化推题，因材施教 |
| 错题没人讲 | AI随时答疑，同伴互助 |
| 学困生不敢问老师 | AI没有"情绪"，学生敢问 |

**全国已有200多所学校采用类似模式**，其中以县域民办高中为主——说明这条路，是走得通的。

---

### "三段四化"智慧课堂教学模式（10分钟）

> **来源**：Todo工作室《AI赋能下的智慧课堂"三段四化"教学模式进阶探索》

**"三段"：课前 - 课中 - 课后**

1. **课前（前置学习）**
   - 教师设计前置学习单，推送到学生平板
   - 学生完成自学，提出质疑
   - AI收集学情数据，教师提前了解学生掌握情况

2. **课中（精准教学）**
   - 不再"从头讲到尾"
   - 重点讲学生不懂的、容易错的
   - 分层教学：不同水平的学生，任务不同

3. **课后（个性化巩固）**
   - AI根据每个学生的情况，推送不同的练习题
   - 错题自动整理，针对性复习
   - 教师查看数据，个别辅导

**"四化"：个性化、数据化、智能化、精准化**

> **核心**：不是用AI替代教师，而是用AI让教师"看得见每个学生的状态"。

## 第五部分：课堂教学怎么改——具体可操作的方法

### 方法一：培养学生精准提问能力（5分钟）

**每节课加入"好问题时间"（最后5分钟）：**

- 规则：不接受"是不是""对不对"这类问题
- 鼓励学生提出：为什么？如果不这样会怎样？这个和那个有什么区别？
- 教师示范：先提出一个"好问题"，让学生模仿

**用AI训练提问能力：**

- 让学生用AI回答问题，然后评价：AI的回答好不好？
- 如果不好，是哪个问题没问清楚？——反过来训练提问能力

---

### 方法二：麻省理工学霸AI学习法（10分钟）

> **来源**：Todo工作室《麻省理工学霸AI学习法，2天学完一门课》
> **B站链接**：https://www.bilibili.com/video/BV1PMQnBFE9K/

**核心方法：三问法（先搭框架，再填细节）**

| 步骤 | 问题 | 目的 |
|------|------|------|
| 第一问（10分钟） | 这个领域专家共有的5个核心思维模型是什么？ | 掌握学科"骨架"，而非死记硬背 |
| 第二问（10分钟） | 专家们存在根本性分歧的3个地方是什么？ | 理解学科前沿与争议，培养批判性思维 |
| 第三问（10分钟） | 能区分"深度理解者"和"单纯记忆者"的10个问题是什么？ | 用高难度问题倒逼深度理解 |

**课堂实操：**

- 不再"先讲定义再讲例题"
- 而是"先让学生用AI搭建知识框架，再来精细化理解"
- 教师的作用是：设计问题、引导讨论、纠正偏差

### 方法三：给孩子选择权，激发内驱力（10分钟）

> **来源**：Todo工作室《提升孩子内驱力——多给孩子一些选择权》

**理论依据：自我决定理论（SDT）**

```
自主权 → 胜任感 → 归属感
（选择权是内驱力的源头）
```

**四大内在机制：**

| 给予选择 | 激发 |
|---------|------|
| 掌控感 | 责任感 |
| 适合难度 | 成功体验 |
| 自主规划 | 目标感 |
| 多元路径 | 发现热爱 |

**阜南亲情学校实践（5个维度）：**

1. **个性化学习进度**——可慢一到两章，学生自主安排节奏
2. **五档难度分层**——不搞全班一套题，匹配不同水平
3. **"零班"模式**——无课表、无讲台、无全程授课（选择权的极致）
4. **学生自主管理委员会**——学生参与学校治理
5. **游学拓展**——走出去看见更大的世界

**行动建议：**

| 层级 | 具体做法 |
|------|---------|
| 立即能做 | 今天留一个选项让学生选（如"先做选择题还是大题？"） |
| 进阶做法 | 分层作业设计 + 学生自主规划表 |
| 教师角色转变 | 从"管控者"到"学习设计师" |

**落地提醒：**
- 避免误区：**选择权≠放任自流**
- 在教师设计的框架内，给学生自主的空间
- 今天就开始：给学生一个"可以选择"的机会

**视频参考：** https://www.bilibili.com/video/BV1xDMv6eE4w/

### 方法四：从"讲知识"到"练能力"（5分钟）

**重新设计课堂时间分配：**

```
传统课堂（45分钟）：
  老师讲 35分钟 + 学生练 10分钟

AI时代课堂（45分钟）：
  前置自学 0分钟（课前完成）
  精讲重点 15分钟（只讲学生不懂的）
  能力训练 20分钟（讨论、合作、创造）
  总结反思 10分钟
```

> **核心**：把"知识传递"放到课前（用AI辅助），把"课堂时间"用在最有价值的地方——能力训练。
